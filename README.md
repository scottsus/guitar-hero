# Guitar Hero

## Usage
- Open Vivado
- Open project(vga_demo.xpr)
- Generate Bitstream
  - it automatically runs Synthesis and Implementation
- Open Hardware Manager
  - open target
  - program device(whateverfile.bit)
  
### Additional Info

to connect your monitor to your NEXYS4.
